# GraphQl

Aplicação criada para aplicar os conceitos básicos do GraphQl;

## Conceitos aplicados
- O que é o GraphQl;
- Quais problemas resolve;
- Operações:
    - Querys;
    - Mutations.
- Modelos de escrita:
    - Schema first aproach;
    - Code first.

## Como executar a aplicação:
### Schema first aproach:

Execute:
```bash
npm run dev:simple
```

### Code first:

Execute:
```bash
npm run dev
```
